clarity by tobi

Web interface for the grep and tail -f unix tools. Useful for real-time log analysis.

Clarity is a web interface for your server log files. It supports searching (using grep) as well as trailing log files in realtime. It has been written using the event based architecture based on EventMachine and so allows real-time search of very large log files. If you hit the browser Stop button it will also kill the grep / tail utility. We wrote Clarity to allow our support staff to use a simple interface to look through the various log files in our server farm. The application was such a big success internally that we decided to release it as open source.

Dependencies

eventmachine eventmachine_httpserver json

Install

sudo gem install clarity
